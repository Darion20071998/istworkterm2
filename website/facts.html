How does HTML work?

HTMl has many attributes and tags which help the computer display the requested content. These can range from tags which can change the look of your page, to adding interactive mediums. All of these tags are linked to a .html and .css file, which is downloaded to a machine's memory everytime you visit a website.

What are the basic tags of HTML?

All HTML tags have a start and a finish. This is <___> and < /___>s . The closing tag has a backslash behind the tag name.

P - The P tag (Paragraph) is used to display standard text to a webpage. Most smaller and body areas are comprised of the P tags. MDN Doc

A - The A tag (Anchor) tag is used to display a hyperlink in a webpage. And nearly all buttons and links (Except for javascript ones) are anchor tags. MDN Doc

img - The img tag (Image) is used to display images in a website, and has a specific structure. The structure is: < img src="IMAGESOURCE"> MDN Doc

strong - A strong is a bold form of text and has the same properties of a p tag. MDN Doc

What is CSS?

CSS (Cascading Style Sheets) is a styling language for HTML and webpages. You are able to change everything related to a webpage. MDN Doc
